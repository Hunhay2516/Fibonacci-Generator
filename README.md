# Fibonacci Generator

## Description

This project is a TypeScript console application that generates Fibonacci numbers based on the number of terms entered by the user. The program uses a class to organize the logic, validates user input, calculates Fibonacci values with recursion, and displays both the individual terms and the full list of generated values. The purpose of the project is to practice TypeScript syntax, classes, arrays, tuples, recursion, and user input.

## Instructions for Build and Use

### Steps to build and/or run the software:

1. Clone or download the repository from GitHub.
2. Open the project folder in Visual Studio Code.
3. Install dependencies with:
   `npm install`
4. Compile the TypeScript file with:
   `tsc`
5. Run the program with:
   `node main.js`

### Instructions for using the software:

1. Start the program in the terminal.
2. Enter the number of Fibonacci terms you want to generate.
3. View the generated Fibonacci sequence in the console.
4. If invalid input is entered, read the error message and try again.
